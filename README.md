# Helios Intelligence

AI-Powered Relationship Intelligence Platform

## Overview

Helios Intelligence helps users discover the most valuable connections for a specific business objective. The platform ingests lead data, enables search and filtering, and will eventually use AI to rank and recommend the most relevant contacts.

## Current Features

### Phase 1 - Lead Management Foundation

* FastAPI Backend
* PostgreSQL Database
* Dockerized Infrastructure
* Lead Creation API
* Pagination
* Search
* Filtering
* Repository Pattern
* Service Layer Architecture
